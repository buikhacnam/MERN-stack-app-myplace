# My Place Mern App

##  https://buinam-myplace.netlify.app/ or run the app locally:

# Frontend 

### `cd frontend`
Go to directory frontend

### `yarn`
Install dependencies

### `yarn start`
Run the app in the port 3000

# Backend 

### `cd backend`
Go to directory backend

### `yarn`
Install dependencies

### `yarn start`
Run the app in the port 5000
